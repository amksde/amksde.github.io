| Action                       | IntelliJ Linux / Windows  | IntelliJ MacOs    |
|------------------------------|---------------------------|-------------------|
| Search Everywhere            | Double Shift              | Double Shift      |
| Intent Actions               | Alt + Enter               | Opt + Enter       |
| Recent Files Popup           | Ctrl + E                  | Cmd + E           |
| Rename                       | Shift + F6                | Shift + F6        |
| Maximize Editor Window       | Ctrl + Shift + F12        | Cmd + Shift + F12 |
| Open IDE Settings            | Ctrl + Alt + S            | Cmd + ,           |
| Open Project Settings        | Ctrl + Alt + Shift + S    | Cmd + ;           |
| Find                         | Ctrl + F                  | Cmd + F           |
| Find In Files                | Ctrl + Shift + F          | Cmd + Shift + F   |
| Replace In Files             | Ctrl + Shift + R          | Cmd + Shift + R   |
| Basic Code Completion        | Ctrl + Space              | Cmd + Space       |
| Optimize Imports             | Ctrl + Alt + O            | Ctrl + Opt + O    |
| Expand / Collapse Code Block | Ctrl + `+`                | Cmd + `+`         |
| Expand All                   | Ctrl + Shift + `+`        | Cmd + Shift + `+` | 
| Collapse All                 | Ctrl + Shift + `-`        | Cmd + Shift + `-` | 
| Go To Declaration            | Ctrl + B                  | Cmd + B           |
| Go To Implementation         | Ctrl + Alt + B            | Cmd + Shift + B   |
| Move Back / Forward          | Ctrl + Alt + Left / Right | Cmd + `[` / `]`   |
| File Structure Popup         | Ctrl + F12                | Cmd + F12         |
